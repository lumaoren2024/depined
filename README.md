# Depined Bot 节点挂机脚本
## 有哪些功能
1、自动挂机节点，30秒发送一次心跳给服务器！
2、防cloudflare拦截功能
3、支持socks5和http 2种代理协议。使用办法：socks5://u:p@ip:port or http://u:p@ip:port

注册地址：https://app.depined.org/dashboard
