## Depined Bot 3.0 教程
1、``accounts.json``文件是用户存储文件
```json
[
    {
        "Email": "001@gmail.com",
        "Password": "dandan"
    },
    {
        "Email": "002@gmail.com",
        "Password": "dandan"
    }
]
```

2、``register.json``文件是注册文件
```json
[
    {
        "Email": "001@gmail.com",
        "Password": "dandan"
    },
    {
        "Email": "002@gmail.com",
        "Password": "dandan"
    }
]
```

3、``proxy.txt``是代理格式
